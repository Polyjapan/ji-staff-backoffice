Utilisez le formulaire suivant pour ajouter des capacités aux staffs. Merci de suivre le format suivant :

Numéro de staff,Capacité 1,Capacité 2,Capacité 3

Par exemple :

  1,Anglais,Espagnol
  2,Fort
  3,Electricien,Anglais,Japonais

Les staffs non renseignés n'ont aucune capacité.
